# DataInsight AI - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
DataInsight AI - 智能数据分析平台

### 1.2 产品定位
面向非技术人员的 AI 驱动数据分析工具，通过自然语言交互降低数据分析门槛，让每个人都能轻松获取数据洞察。

### 1.3 目标用户
- **核心用户**：产品经理、运营人员、业务分析师
- **次要用户**：中小企业管理者、数据新手

### 1.4 核心价值主张
- **零门槛**：无需 SQL 知识，自然语言即可查询
- **高效率**：秒级响应，即时获取分析结果
- **智能化**：AI 自动生成洞察建议，辅助决策

---

## 2. 需求分析

### 2.1 用户痛点
| 痛点 | 描述 | 影响 |
|------|------|------|
| 技术门槛高 | 数据分析需要掌握 SQL 等技能 | 业务人员无法自主分析数据 |
| 效率低下 | 每次分析需求需要找技术团队 | 决策周期长，错过最佳时机 |
| 洞察不足 | 看到数据但不知如何解读 | 数据价值无法充分发挥 |

### 2.2 解决方案
- **自然语言转 SQL**：AI 理解用户意图，自动生成查询语句
- **可视化展示**：自动生成图表，直观呈现数据
- **智能洞察**：AI 分析数据趋势，提供业务建议

---

## 3. 功能设计

### 3.1 核心功能

#### 3.1.1 自然语言查询
- **输入**：用户用中文描述数据需求
- **处理**：AI 解析意图，生成 SQL
- **输出**：返回查询结果

**示例**：
```
输入："查询2024年各月份的销售额"
输出：SQL + 折线图 + 趋势分析
```

#### 3.1.2 数据可视化
- 自动识别数据类型，推荐最佳图表
- 支持折线图、柱状图、饼图等
- 响应式设计，适配各种屏幕

#### 3.1.3 AI 数据洞察
- 自动识别数据趋势和异常
- 生成业务解读和建议
- 支持多维度对比分析

### 3.2 辅助功能

#### 3.2.1 查询历史
- 保存用户查询记录
- 支持快速重新执行
- 便于追溯和对比

#### 3.2.2 示例引导
- 提供常用查询模板
- 帮助用户快速上手
- 展示产品能力边界

---

## 4. 技术架构

### 4.1 技术栈
- **前端**：Vue 3 + Vite + Canvas API
- **后端**：Node.js + Express
- **AI 能力**：智谱 AI GLM-4-Flash
- **部署**：Gitee Pages + Render

### 4.2 系统架构
```
用户输入 → 前端界面 → 后端 API → AI 模型
                ↓           ↓
            展示结果 ← 生成 SQL + 洞察
```

### 4.3 数据流
1. 用户输入自然语言查询
2. 后端调用 AI 生成 SQL
3. 执行 SQL（模拟数据）
4. 生成可视化图表
5. AI 生成数据洞察
6. 返回完整结果

---

## 5. 产品亮点

### 5.1 创新点
- **Text-to-SQL 落地应用**：将前沿 AI 技术产品化
- **端到端解决方案**：从查询到洞察的完整闭环
- **零成本使用**：基于免费 AI API，降低使用门槛

### 5.2 竞争优势
| 维度 | 传统 BI 工具 | DataInsight AI |
|------|-------------|----------------|
| 学习成本 | 高（需学 SQL） | 零（自然语言） |
| 部署成本 | 高（需购买） | 零（免费使用） |
| 响应速度 | 分钟级 | 秒级 |
| 智能程度 | 被动查询 | 主动洞察 |
